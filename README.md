# Student Registration System

## Objective
The objective of this project is to create a Student Registration System where students can register their details.

## Features
- Add student details
- Edit student records
- Delete student records
- Data persistence using localStorage
- Input validation
- Responsive design

## Technologies Used
- HTML
- Tailwind CSS
- JavaScript

## How to Run
Open the index.html file in any modern browser.
